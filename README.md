# Marz Software

Landing page para **Marz Software** - una empresa de desarrollo de software a medida.

## Sobre el Nombre

**Marz** combina un apellido con un juego de palabras con **Mars** (Marte), simbolizando innovación, nuevos horizontes y la exploración de nuevas fronteras digitales.

## Colores de Marca

- **#000** (Negro) - Texto principal
- **#e94e1b** (Naranja/Rojo) - Acentos, bordes, detalles
- **#FFF** (Blanco) - Fondo

## Tecnologías

- [Next.js](https://nextjs.org/) - Framework de React
- [Tailwind CSS](https://tailwindcss.com/) - Framework de CSS
- [TypeScript](https://www.typescriptlang.org/) - Tipado estático

## Desarrollo

```bash
# Instalar dependencias
npm install

# Iniciar servidor de desarrollo
npm run dev

# Compilar para producción
npm run build

# Iniciar servidor de producción
npm start
```

Abre [http://localhost:3000](http://localhost:3000) en tu navegador para ver el resultado.

## Estructura

```
src/
├── app/
│   ├── globals.css    # Estilos globales y colores
│   ├── layout.tsx     # Layout principal
│   └── page.tsx       # Landing page
```
